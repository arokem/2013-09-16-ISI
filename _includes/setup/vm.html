Virtual Machine

Installation issues can and do happen. To ensure that you can continue to participate in a lesson even if one of your software programs fails, we provide a Linux virtual machine that contains most of the necessary software pre-installed.

Please download and install Virtual Box for your OS from here . On the first day we will do a walkthrough of how to setup and launch the bootcamp VM through Virtual Box.

The Workflows module and the RNASeq Module on the second day will be done using the bootcamp VM.

Note: The VM does not have a GUI. The IPython notebook and the plotting exercises with matplotlib need X environment and cannot be done in the VM.
